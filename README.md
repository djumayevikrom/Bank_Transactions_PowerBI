# Bank_Transactions_PowerBI
This project analyzes bank customer transactions using Power BI.
The dashboard provides visual insights into transaction patterns, customer spending behavior, cash flow movements, and category-based trends.
Key KPIs (such as total transaction amount, average transaction size, category contribution, and monthly trend changes) are calculated using DAX formulas.

# Dataset Structure
The dataset includes the following key columns:
- TransactionID
- CustomerName
- Category
- Amount
- TransactionDate
- Branch
- Currency
- TransactionType
- AccountStatus

# Project Objective
- Visualize spending trends and category contributions
- Calculate key financial KPIs using DAX
- Compare performance over time (yearly)

# Key KPIs (DAX-based)
- Total Transaction Amount
- Average Transaction Amount
- Category Contribution
- Branch-wise Performance

# Tools Used
- Power BI
- DAX
- Excel (dataset preparation)

# Data Note
All data is anonymized and used for portfolio purposes.
